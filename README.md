# Credit Card Portfolio Analysis

## Overview
End-to-end Power BI project analyzing credit card portfolio performance using customer and transaction data, focused on revenue, utilization, delinquency indicators, and customer behavior.

## Business Problem
How can customer and transaction-level analytics support better portfolio monitoring, profitability analysis, and business decisions?

## Tools Used
- Power BI
- SQL
- DAX

## Analysis Areas
- Customer segmentation analysis
- Transaction performance monitoring
- Credit utilization analysis
- Delinquency indicator tracking
- Revenue and interest earned analysis

## Key KPIs
- Total Revenue
- Total Transactions
- Total Transaction Amount
- Interest Earned
- Utilization Ratio
- Delinquency Metrics

## Key Insights
- Identified high-revenue customer/card segments
- Analyzed transaction trends across quarters
- Evaluated profitability and utilization patterns
- Monitored delinquency-related risk indicators

## Repository Structure
- data/ → source datasets
- dashboards/ → dashboard screenshots
